■ ⚠ 地雷（接手務必守）
  1. GitHub 是唯一真相；一律在 git 工作樹工作、文件跟程式一起 commit；動手前先確認在哪個分支。
  2. 併 main = Netlify 自動部署（保羅/大廳/約拿正式站）。動作關原型別 push 約拿 main（會曝光）。
  3. 牧者把關：盼望/大光「卡片版」文案未過審別連到大廳；新產的題/文案一律由牧者審。
  4. .bat 純英文 ASCII + CRLF；啟動 `npm run dev -- --open "/?demo=KEY"`；別寫 Unix 的 >/dev/null，要 nul。
  5. 卡片 spec 的 answer/lives 不可越界；過 game-content-validator / npm test。
  6. agape250 那台 vite build 偶爾無聲死（exit 127）：先手動刪 dist/ 再 build。HFP 那台無此問題；Netlify(Node 20)不受影響。
  7. 零美術檔 = 改完場景一定截圖（保羅）或跑 headless 渲染檢查（約拿）驗收。
