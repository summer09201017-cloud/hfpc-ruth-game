  4. 巴比倫的城牆倒塌
📖 但 7:9-10｜我觀看，見有寶座設立，上頭坐著亙古常在者……事奉他的有千千，在他面前侍立的有萬萬。
💡 地上再亂，天上的寶座從來沒有空過。審判者坐著要行審判，案卷都展開了。
☐ 步驟 3：「有一位像人子的，駕著天雲而來」——祂得了什麼？
  1. 一座新的王宮
  2. ✅ 權柄、榮耀、國度，使各方、各國、各族的人都事奉他
  3. 巴比倫的王位
  4. 四獸的領土
📖 但 7:13-14｜得了權柄、榮耀、國度，使各方、各國、各族的人都事奉他。他的權柄是永遠的，不能廢去。
💡 主耶穌最喜歡用「人子」稱呼自己——這一幕指向祂。祂的國不像獸的國會過去，乃是永遠的。
☐ 步驟 4：到了結局，國度、權柄要歸給誰？
  1. 最強的帝國
  2. ✅ 至高者的聖民
  3. 最聰明的哲士
  4. 巴比倫的後代
📖 但 7:27｜國度、權柄，和天下諸國的大權必賜給至高者的聖民。他的國是永遠的。
💡 屬神的人在世上常像被擄的但以理——但結局已經寫好了：國度是聖民的。所以可以忠心到底。
☐ 步驟 5：但以理是從哪位先知的書上，得知耶路撒冷荒涼七十年將滿？
  1. 以賽亞
  2. ✅ 耶利米
  3. 以西結
  4. 阿摩司
📖 但 9:2；12:13｜我但以理從書上得知耶和華的話臨到先知耶利米……「你且去等候結局，因為你必安歇。」
💡 讀經帶出禱告，禱告迎接應許——古列元年，歸回的詔令果然發出。而神給老但以理的最後一句話是：你且去等候結局，你必安歇。
☐ 結尾：✓ 全書終 · 神掌權
從被擄的少年到白髮的總長，但以理用一生見證了一句話：「至高者在人的國中掌權。」（但 4:17）王朝換了四個，他事奉的神從未改變。

🐸 十災的對決 (tenPlagues)

☐ 開場：🐸 十災的對決
📖 出埃及記 7:5｜我伸手攻擊埃及，將以色列人從他們中間領出來的時候，埃及人就要知道我是耶和華。
法老說：「耶和華是誰，使我聽他的話？」（出 5:2）神就用十災回答他。請把十災由先到後排出來——錯了沒關係，排完你就把整個故事記住了！
☐ 步驟 1（依序點選）：從第一災開始，依序點選：
  1. ✅ 🩸 水變血
  2. ✅ 🐸 青蛙
  3. ✅ 🦟 虱子
  4. ✅ 🪰 蒼蠅
  5. ✅ 🐄 畜疫
  6. ✅ 🤕 瘡
  7. ✅ 🌨️ 冰雹
  8. ✅ 🦗 蝗蟲
  9. ✅ 🌑 黑暗
  10. ✅ 💔 擊殺長子
📖 出 7–12 章｜法老的心剛硬，不肯容百姓去……直到那一夜，埃及全地大大哀號。
💡 前九災一次次給法老回轉的機會，他卻一再剛硬；第十災臨到，他終於容百姓去。神有大能，也有忍耐。
☐ 步驟 2：十災攻擊的對象，其實是誰？
  1. 埃及的百姓
  2. ✅ 埃及一切的神（假神）
  3. 埃及的軍隊
  4. 尼羅河
📖 出 12:12｜我要敗壞埃及一切的神。我是耶和華。
💡 尼羅河、青蛙、太陽……都是埃及人拜的神明。十災逐一顯明：唯有耶和華是真神，祂為祂的百姓爭戰。
☐ 步驟 3：擊殺長子的那一夜，以色列人的家靠什麼得保護？
  1. 緊鎖的門閂
  2. ✅ 門框和門楣上羔羊的血
  3. 屋頂上的記號
  4. 整夜的守望
📖 出 12:13｜這血要在你們所住的房屋上作記號；我一見這血，就越過你們去。
💡 救贖是靠羔羊的血——逾越節由此而來，也指向那位「神的羔羊，除去世人罪孽的」（約 1:29）。
☐ 結尾：✓ 容我的百姓去！
當夜法老召了摩西、亞倫來，說：「起來！連你們帶以色列人，從我民中出去。」（出 12:31）四百三十年的為奴歲月，在這一夜結束了。

📜 西奈山 · 十誡 (tenCommandments)

☐ 開場：📜 西奈山 · 十誡
📖 出埃及記 19:18｜西奈全山冒煙，因為耶和華在火中降於山上……遍山大大地震動。
雷轟、閃電、密雲、角聲——神親口吩咐十條誡命。這是立約的時刻，每個人都要在山下站立領受。
☐ 步驟 1：神頒布十誡之前，先說了哪一句話？
  1. ✅ 「我是耶和華你的神，曾將你從埃及地為奴之家領出來。」
  2. 「你們要竭力遵守誡命。」
  3. 「天地都是我造的。」
  4. 「摩西是我的僕人。」
📖 出 20:2｜我是耶和華你的神，曾將你從埃及地為奴之家領出來。
💡 十誡的開頭不是命令，而是救恩的宣告——神先救贖、後吩咐。我們守誡命不是為了得救，而是因為已蒙拯救。
☐ 步驟 2：十誡的第一條是什麼？
  1. 不可殺人
  2. ✅ 除了我以外，你不可有別的神
  3. 當孝敬父母
  4. 當記念安息日
📖 出 20:3｜除了我以外，你不可有別的神。
💡 十誡先講人與神的關係（1–4 誡），再講人與人的關係（5–10 誡）——愛神，然後愛人。
☐ 步驟 3：有人說：「拜拜偶像求平安沒關係，心裡信耶穌就好。」這違背了哪一條誡命？
  1. ✅ 第二誡：不可為自己雕刻偶像……不可跪拜那些像
  2. 第四誡：當記念安息日
  3. 第八誡：不可偷盜
  4. 第六誡：不可殺人
📖 出 20:4-5｜不可為自己雕刻偶像……不可跪拜那些像，也不可事奉它。
💡 剛領受完十誡，山下就鑄了金牛犢（出 32）——人心多麼容易為神「另做一個樣子」。敬拜只歸獨一的真神。
☐ 步驟 4：爸媽請你幫忙，你翻白眼、不耐煩地頂嘴。這提醒我們哪一條誡命？
  1. 第三誡：不可妄稱耶和華你神的名
  2. ✅ 第五誡：當孝敬父母
  3. 第九誡：不可作假見證
  4. 第十誡：不可貪戀
📖 出 20:12｜當孝敬父母，使你的日子在耶和華你神所賜你的地上得以長久。
💡 這是第一條帶應許的誡命（弗 6:2）。孝敬不只是聽話，更是從心裡敬重。
☐ 步驟 5：同學有最新的玩具，你心裡一直想：「為什麼他有我沒有……」這是哪一條誡命提醒的事？
  1. 第七誡：不可姦淫
  2. 第八誡：不可偷盜
  3. ✅ 第十誡：不可貪戀人的房屋……並他一切所有的
  4. 第一誡：不可有別的神
📖 出 20:17｜不可貪戀人的房屋；也不可貪戀人的妻子、僕婢、牛驢，並他一切所有的。
💡 十誡的最後一條管到「心裡」——還沒做出來、只是貪想，神已經看見。誡命帶我們承認：我們需要救主。
☐ 結尾：✓ 立約之民
「你們要歸我作祭司的國度，為聖潔的國民。」（出 19:6）先有出埃及的救贖，才有西奈山的誡命——先恩典，後律法。

⛺ 會幕榮光 · 五幕 (exodusFinale)

☐ 開場：⛺ 會幕榮光 · 終局五幕
📖 出埃及記 25:8｜又當為我造聖所，使我可以住在他們中間。
出埃及的終點不是一塊土地，而是「神住在祂百姓中間」。但在會幕立起來之前，先發生了一場悲劇……請看五幕。
☐ 步驟 1：摩西在山上遲延不下來，百姓做了什麼？
  1. 安靜等候四十天
  2. ✅ 鑄了一隻金牛犢說「這是領你出埃及的神」
  3. 派約書亞上山找摩西
  4. 收拾行李回埃及
📖 出 32:1-4｜他們就說：「以色列啊，這是領你出埃及地的神！」
💡 立約的墨跡未乾，山下已拜起金牛犢。不是不信「有神」，而是擅自把神改成自己喜歡的樣子。
☐ 步驟 2：神的怒氣發作，摩西在山上做了什麼？
  1. 同意神除滅百姓
  2. ✅ 懇求耶和華，提起神對亞伯拉罕、以撒、以色列起的誓
  3. 默默下山不說話
  4. 為自己求一個更大的國
📖 出 32:11-14｜摩西便懇求耶和華他的神……於是耶和華後悔，不把所說的禍降與他的百姓。
💡 摩西抓住神自己的應許代求——這是中保的身影，指向那位「長遠活著、替我們祈求」的主（來 7:25）。
☐ 步驟 3：摩西求「顯出祢的榮耀給我看」，神在他面前宣告了什麼？
  1. ✅ 「耶和華，耶和華，是有憐憫有恩典的神，不輕易發怒，並有豐盛的慈愛和誠實。」
  2. 「我是烈火，不可靠近。」
  3. 「你配不上看見榮耀。」
  4. 「以後再說。」
📖 出 34:6-7｜耶和華，耶和華，是有憐憫有恩典的神，不輕易發怒，並有豐盛的慈愛和誠實。
💡 金牛犢之後，神顯出的榮耀竟是祂的「名」——憐憫、恩典、不輕易發怒。罪之後還有恩典，這就是福音的預告。
☐ 步驟 4：會幕的一切工程做完，是照著什麼做成的？
  1. 摩西自己的設計
  2. ✅ 耶和華所吩咐摩西的一切（山上指示的樣式）
  3. 埃及神廟的圖樣
  4. 百姓投票的結果
📖 出 40:16；25:9｜摩西這樣行，凡耶和華所吩咐他的，他都照樣行了。
💡 敬拜的方式由神定規，不由人發明——這正是金牛犢學到的功課。照神所吩咐的，祂就臨近。
☐ 步驟 5：會幕立起來之後，什麼充滿了帳幕？
  1. 馨香的煙
  2. ✅ 耶和華的榮光
  3. 百姓的歌聲
  4. 獻祭的牛羊
📖 出 40:34-38｜當下，雲彩遮蓋會幕，耶和華的榮光就充滿了帳幕。
💡 從此以色列人一切行路的事上，都有耶和華的雲彩引導。神救我們出來，是要住在我們中間——「道成了肉身，住在我們中間」（約 1:14）。
☐ 結尾：✓ 全書終 · 神的同在
從為奴之家到會幕榮光：救贖（羔羊的血）→ 立約（十誡）→ 同在（會幕）。出埃及的目的地，是神自己。
